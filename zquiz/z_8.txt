Вопросы по табличным пространствам и каталогам PostgreSQL
1. Какой командой создается табличное пространство в PostgreSQL?
a) CREATE TABLESPACE
b) CREATE SPACE
c) ADD TABLESPACE
d) NEW TABLESPACE
Правильный ответ: a

2. В каком системном каталоге хранится информация о табличных пространствах?
a) pg_tablespace
b) pg_space
c) pg_storage
d) pg_location
Правильный ответ: a

3. Какой системный каталог содержит информацию о всех таблицах базы данных?
a) pg_class
b) pg_tables
c) pg_objects
d) pg_relations
Правильный ответ: a

4. Как посмотреть список всех табличных пространств в кластере?
a) SELECT * FROM pg_tablespace
b) SHOW TABLESPACES
c) LIST TABLESPACES
d) DISPLAY TABLESPACES
Правильный ответ: a

5. Какой параметр команды CREATE DATABASE указывает табличное пространство для новой базы данных?
a) LOCATION
b) TABLESPACE
c) DIRECTORY
d) DATASPACE
Правильный ответ: b

6. В каком каталоге по умолчанию хранятся данные кластера PostgreSQL?
a) DATA_DIR
b) PGDATA
c) CLUSTER_DIR
d) POSTGRES_DATA
Правильный ответ: b

d) pg_catalog_db
