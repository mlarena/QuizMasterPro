Управление программным обеспечением в Astra Linux
1. Какой формат пакетов используется в Astra Linux?
a) RPM
b) DEB
c) TGZ
d) TXZ
Правильный ответ: b

2. Какой командой установить локальный пакет program.deb?
a) apt install ./program.deb
b) dpkg -i program.deb
c) apt-get install program.deb
d) Варианты a и b верны
Правильный ответ: d

3. Где находятся основные конфигурационные файлы менеджера пакетов APT?
a) /etc/apt/
b) /var/lib/apt/
c) /usr/share/apt/
d) /etc/dpkg/
Правильный ответ: a

4. Какой командой обновить список доступных пакетов из всех подключенных репозиториев?
a) apt update
b) apt upgrade
c) apt dist-upgrade
d) apt-get update
Правильный ответ: a

5. Какой файл содержит список всех подключенных репозиториев APT?
a) /etc/apt/apt.conf
b) /etc/apt/sources.list
c) /etc/apt/preferences
d) /var/lib/apt/lists/
Правильный ответ: b

6. Какой командой удалить пакет, оставив его конфигурационные файлы?
a) apt remove package
b) apt purge package
c) dpkg -r package
d) Варианты a и c верны
Правильный ответ: d

7. Какой компонент пакета DEB содержит контрольную информацию и скрипты установки?
a) data.tar.*
b) control.tar.*
c) debian-binary
d) metadata
Правильный ответ: b

8. Какой командой просмотреть информацию об установленном пакете nginx?
a) apt show nginx
b) dpkg -s nginx
c) apt-cache show nginx
d) Все перечисленные
Правильный ответ: d

9. Какой командой найти пакет, содержащий файл /usr/bin/python3?
a) dpkg -S /usr/bin/python3
b) apt-file search /usr/bin/python3
c) apt search python3
d) Варианты a и b верны
Правильный ответ: d

10. Какой командой добавить сторонний репозиторий, используя GPG-ключ?
a) add-apt-repository ppa:user/ppa
b) apt-add-repository 'deb [arch=amd64] http://repo.example.com/astra stable main'
c) echo 'deb http://repo.example.com/astra stable main' | sudo tee /etc/apt/sources.list.d/custom.list
d) Варианты b и c верны
Правильный ответ: d

11. Какой утилитой можно создать локальный репозиторий из набора DEB-пакетов?
a) dpkg-scanpackages
b) apt-ftparchive
c) reprepro
d) Любой из перечисленных
Правильный ответ: d

12. Какой командой установить все обновления безопасности?
a) apt upgrade --security
b) apt-get dist-upgrade
c) unattended-upgrade
d) apt safe-upgrade
Правильный ответ: c

13. Какой файл позволяет настроить приоритеты пакетов из разных репозиториев (pinning)?
a) /etc/apt/apt.conf.d/01priority
b) /etc/apt/preferences
c) /etc/apt/sources.pref
d) /etc/dpkg/origins/default
Правильный ответ: b

14. Какой командой просмотреть список установленных пакетов?
a) dpkg -l
b) apt list --installed
c) aptitude search ~i
d) Все перечисленные
Правильный ответ: d

15. Какой командой загрузить пакет и все его зависимости без установки?
a) apt download package
b) apt-get download package
c) apt-get install --download-only package
d) wget package.deb
Правильный ответ: c

16. Какой инструмент предоставляет графический интерфейс для управления пакетами?

d) Варианты a и c верны

18. Какой файл содержит информацию о доступных версиях пакетов в репозиториях?
