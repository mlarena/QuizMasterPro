Основы работы в командной строке ОС Astra Linux
1. Какой из перечисленных терминалов является графическим эмулятором терминала (псевдотерминалом)?
a) tty1 (доступный по Ctrl+Alt+F1)
b) pts/0 (запущенный, например, в GNOME Terminal)
c) console (системная консоль)
d) ttyS0 (последовательный порт)
Правильный ответ: b

2. Какая команда используется для безопасного завершения сеанса текущего пользователя в командной строке?
a) logout
b) exit
c) Ctrl+D (ввод символа конца файла)
d) Все перечисленные варианты верны
Правильный ответ: d

3. Что означает символ ~ (тильда) в начале пути в командной строке?
a) Домашний каталог текущего пользователя
b) Домашний каталог пользователя root
c) Корневой каталог файловой системы
d) Каталог временных файлов
Правильный ответ: a

4. Какая из следующих команд использует "длинный формат" параметров (long options)?
a) ls -l
b) ls --all
c) ls -a
d) ls -la
Правильный ответ: b

5. Как создать переменную окружения MYVAR со значением "test" в сеансе командной оболочки bash?
a) set MYVAR=test
b) MYVAR = "test"
c) MYVAR=test
d) export MYVAR="test"
Правильный ответ: c (Создание переменной. Вариант d также создаст, но и экспортирует в окружение дочерних процессов.)

6. Какое выражение использует подстановку команд (command substitution) для вставки вывода команды date в строку?
a) $(date)
b) `date`
c) %date%
d) Варианты a и b верны
Правильный ответ: d

7. Как выполнить арифметическое вычисление 10 + 5 непосредственно в командной оболочке bash?
a) echo 10 + 5
b) expr 10 + 5
c) echo $((10 + 5))
d) calc 10+5
Правильный ответ: c

8. Какой символ используется для экранирования (отмены интерпретации) специального символа, например, для создания файла с именем "file&name.txt"?
a) / (прямой слеш)
b) ^ (каретка)
c) \ (обратный слеш)
d) ! (восклицательный знак)
Правильный ответ: c

9. Какой символ подстановки (wildcard) соответствует ровно одному любому символу в имени файла?
a) *
b) ?
c) [...]
d) {...}
Правильный ответ: b

10. Как просмотреть последние 15 команд из истории (history) в bash?
a) history 15
b) history | tail -15
c) history -n 15
d) Любой из перечисленных способов
Правильный ответ: d

11. Какая команда создаст постоянный псевдоним ll для команды ls -la в текущей сессии bash?
a) alias ll="ls -la"
b) set ll="ls -la"
c) def ll ls -la
d) ll = ls -la
Правильный ответ: a

12. Какая клавиша используется для автоматического дополнения (автодополнения) имени команды или файла в bash?
a) Tab
b) Enter
c) Space
d) Ctrl
Правильный ответ: a

13. Какой специальный символ позволяет выполнить несколько команд на одной строке, запуская каждую следующую только в случае успеха предыдущей?
a) ; (точка с запятой)
b) & (амперсанд)
c) && (двойной амперсанд)
d) || (двойная вертикальная черта)
Правильный ответ: c

14. Как вывести значение переменной окружения PATH?
a) echo PATH
b) echo $PATH
c) printenv PATH
d) Варианты b и c верны
Правильный ответ: d

15. Что делает конструкция 2>&1 в конце команды, например, command > file.log 2>&1?
a) Перенаправляет стандартный вывод (stdout) в файл, а стандартный ошибок (stderr) — на экран
b) Перенаправляет stderr в то же место, куда направлен stdout (в файл)
c) Запускает команду в фоновом режиме
d) Объединяет вывод двух команд
Правильный ответ: b

16. Как удалить псевдоним ll в текущей сессии bash?
a) unalias ll
b) alias ll=
c) remove alias ll
d) delete alias ll
Правильный ответ: a

17. Какая клавишная комбинация прерывает (отменяет) выполнение текущей команды в терминале?
a) Ctrl+C
b) Ctrl+Z
c) Ctrl+D
d) Ctrl+S
Правильный ответ: a

18. Какая команда выведет список всех переменных окружения?
a) set
b) env или printenv
c) export
d) Все перечисленные (но set выводит также локальные переменные и функции)
Правильный ответ: d

19. Что означает символ # в командной строке bash?
a) Начало комментария
b) Приглашение суперпользователя (root)
c) Оба варианта верны в разных контекстах
d) Специальный символ для поиска по истории
Правильный ответ: c

20. Какой символ подстановки соответствует любому количеству любых символов (включая ноль)?
a) ?
b) *
c) [0-9]
d) %
Правильный ответ: b

21. Как выполнить команду из истории под номером 257, не набирая ее заново?
a) !257
b) history 257
c) run 257
d) #257
Правильный ответ: a

22. Какая конструкция позволяет создать список в фигурных скобках для генерации имён файлов: file1, file2, file3?
a) file{1,2,3}
b) file[1-3]
c) file(1..3)
d) file*
Правильный ответ: a

23. Как сделать переменную MYVAR доступной (экспортировать) для всех дочерних процессов оболочки?
a) set MYVAR=value
b) MYVAR=value; export MYVAR
c) export MYVAR=value
d) Варианты b и c верны
Правильный ответ: d

24. Какой символ используется для перенаправления стандартного ввода (stdin) команды из файла?
a) <
b) >
c) |
d) >>
Правильный ответ: a

25. Что делает клавиша Ctrl+R в интерактивном режиме bash?
a) Перезагружает терминал
b) Запускает обратный поиск (reverse search) по истории команд
c) Перерисовывает экран (redraw)
d) Удаляет слово перед курсором
Правильный ответ: b

26. Какая команда вычисляет выражение 2 * (3 + 5) с помощью утилиты bc (калькулятор)?
a) echo "2 * (3 + 5)" | bc
b) bc <<< "2 * (3 + 5)"
c) echo "2 * (3 + 5)" > bc
d) Варианты a и b верны
Правильный ответ: d

27. Что будет результатом выполнения команды echo \\$HOME?
a) Выведет путь к домашнему каталогу (например, /home/user)
b) Выведет строку $HOME
c) Выведет обратный слеш и путь к домашнему каталогу
d) Выдаст ошибку
Правильный ответ: b

28. Какой символ используется для конвейера (pipe), передающего вывод одной команды на вход другой?
a) >
b) <
c) |
d) &
Правильный ответ: c

29. Как добавить каталог /usr/local/bin в конец переменной PATH на постоянной основе для всех пользователей?
a) Добавить строку PATH=$PATH:/usr/local/bin в файл /etc/environment
b) Добавить строку export PATH=$PATH:/usr/local/bin в файл /etc/profile
c) Выполнить команду export PATH=$PATH:/usr/local/bin в текущей сессии
d) Варианты a и b являются корректными способами
Правильный ответ: d

30. Как вывести на экран все команды из истории, содержащие слово "apt"?
a) history | find "apt"
b) history | grep apt
c) grep apt ~/.bash_history
d) Варианты b и c верны
Правильный ответ: d
