Использование справочных ресурсов в Astra Linux
1. Как получить краткую справку по использованию команды ls, включая список основных опций?
a) man ls
b) info ls
c) ls --help
d) help ls
Правильный ответ: c

2. Какая клавиша используется для прокрутки вниз при просмотре страницы руководства man?
a) Стрелка вверх
b) Пробел или Page Down
c) Enter (на одну строку вниз)
d) Варианты b и c верны
Правильный ответ: d

3. Какой раздел руководства man содержит информацию о системных вызовах (например, open, read)?
a) Раздел 1: Пользовательские команды
b) Раздел 2: Системные вызовы
c) Раздел 5: Форматы файлов
d) Раздел 8: Команды администрирования
Правильный ответ: b

4. Как выйти из просмотрщика страниц man?
a) Нажать клавишу q
b) Нажать Ctrl+C
c) Нажать :q и Enter
d) Нажать Esc
Правильный ответ: a

5. Что делает команда whatis, например, whatis ls?
a) Показывает полное руководство по команде
b) Выводит краткое описание команды из базы данных whatis
c) Ищет команду в каталогах из PATH
d) Показывает, где расположен исполняемый файл команды
Правильный ответ: b

6. Как искать ключевое слово во всех разделах руководств man?
a) man -k keyword или apropos keyword
b) man --search keyword
c) find /usr/share/man -name "*keyword*"
d) grep -r keyword /usr/share/man
Правильный ответ: a

7. Как перейти к следующему вхождению искомого слова при просмотре страницы man (после запуска поиска через /)?
a) Нажать n (next)
b) Нажать N (для поиска в обратном направлении)
c) Нажать f (forward)
d) Нажать пробел
Правильный ответ: a

8. Какая команда отобразит все доступные разделы руководства man для команды printf?
a) man -a printf
b) man -f printf
c) man -w printf
d) man -k printf
Правильный ответ: b

9. Как получить справку по встроенной команде оболочки bash, например, cd?
a) man cd
b) cd --help
c) help cd
d) info cd
Правильный ответ: c

10. Какой формат справочной системы является более подробным и структурированным по сравнению с man, но менее распространенным?
a) --help для всех команд
b) doc файлы в /usr/share/doc
c) info
d) whatis
Правильный ответ: c

11. Как переместиться на один символ вперед в программе просмотра info?
a) Стрелка вправо
b) Ctrl+f
c) Ctrl+b (назад)
d) Варианты a и b верны
Правильный ответ: d

12. Где обычно хранятся локально установленные страницы руководств man?
a) /usr/local/share/man
b) /usr/share/man
c) ~/.man
d) /var/cache/man
Правильный ответ: a

13. Как обновить базу данных для команд whatis и apropos?
a) sudo updatedb
b) sudo makewhatis или sudo mandb
c) sudo man --update
d) sudo apt update man-db
Правильный ответ: b

14. Как просмотреть только описание (синопсис) раздела руководства, без открытия полной страницы?
a) man -f command
b) man -d command
c) man -s command
d) man -l command
Правильный ответ: a

15. Какой раздел man содержит описание форматов конфигурационных файлов, например, fstab?
a) Раздел 1
b) Раздел 4
c) Раздел 5
d) Раздел 7
Правильный ответ: c

16. Как в программе info перейти к следующему узлу (next node)?
a) n
b) p (previous node)
c) u (up)
d) Enter
Правильный ответ: a

17. Что делает команда which, например, which ls?
a) Показывает справку по команде
b) Показывает, какая версия команды будет выполнена (путь к исполняемому файлу)
c) Ищет команду в справочных страницах
d) Определяет, является ли команда встроенной в оболочку
Правильный ответ: b

18. Как отобразить путь к файлу руководства man для команды passwd без его вывода на экран?
a) man -w passwd
b) whereis -m passwd
c) locate passwd.1.gz
d) Все перечисленные варианты могут сработать
Правильный ответ: d

19. Как в info перейти к меню (оглавлению) документа?
a) m
b) t (к началу документа)
c) d (к директории info)
d) l (вернуться назад)
Правильный ответ: a

20. Какая опция команды man позволяет отображать руководство на русском языке, если оно установлено?
a) man -L ru
b) man -l ru
c) LANG=ru_RU.UTF-8 man
d) man --locale=ru
Правильный ответ: c

21. Какой раздел man содержит описание драйверов устройств и специфичных для аппаратуры файлов?
a) Раздел 2
b) Раздел 4
c) Раздел 6
d) Раздел 9
Правильный ответ: b

22. Как в программе info выйти?
a) q
b) Ctrl+C
c) Esc
d) :q
Правильный ответ: a

23. Что делает команда type, например, type cd?
a) Показывает тип команды (встроенная, внешняя, псевдоним)
b) Определяет MIME-тип файла
c) Ищет команду в базе данных whatis
d) Показывает раздел руководства для команды
Правильный ответ: a

24. Как отобразить все встроенные команды оболочки bash?
a) help
b) man builtins
c) compgen -b
d) Варианты a и c верны
Правильный ответ: d

25. Как перейти к определенному разделу в руководстве man, например, к разделу 5 о формате файла passwd?
a) man 5 passwd
b) man -s5 passwd
c) man passwd.5
d) Любой из перечисленных
Правильный ответ: d

26. Как в программе info искать текст?
a) / и затем ввести строку поиска
b) Ctrl+s
c) s
d) Варианты a и b верны
Правильный ответ: d

27. Какая переменная окружения определяет кодировку для отображения страниц man?
a) MANPATH
b) LANG или LC_ALL
c) MANCODEPAGE
d) PAGER
Правильный ответ: b

28. Что делает опция -a в команде man?
a) Показывает все подходящие страницы руководства по очереди
b) Ищет ключевое слово во всех разделах
c) Показывает краткую справку
d) Автоматически определяет раздел
Правильный ответ: a

29. Какой программой-пейджером по умолчанию отображаются страницы man в Astra Linux?
a) more
b) less
c) most
d) pg
Правильный ответ: b

30. Где можно найти подробную документацию по программному пакету в виде текстовых файлов, примеров конфигурации и лицензии?
a) В каталоге /usr/share/doc/имя_пакета
b) На страницах info
c) В руководствах man
d) Во всех перечисленных местах
Правильный ответ: d
