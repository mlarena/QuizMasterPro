Дискреционное управление доступом
1. Какой класс пользователей в системе прав доступа Linux обозначается символом 'o' (others)?
a) Владелец файла
b) Члены группы-владельца файла
c) Все остальные пользователи
d) Только пользователь root
Правильный ответ: c

2. Какое числовое значение соответствует правам доступа 'rwxr-xr--'?
a) 755
b) 754
c) 644
d) 751
Правильный ответ: b

3. Что означает право выполнения (x) для каталога в Linux?
a) Возможность удалять файлы в каталоге
b) Возможность читать содержимое файлов в каталоге
c) Возможность входить в каталог (делать его текущим)
d) Возможность изменять права доступа каталога
Правильный ответ: c

4. Какой специальный бит, установленный на каталоге, запрещает удаление файлов не их владельцам?
a) SUID (Set User ID)
b) SGID (Set Group ID)
c) Sticky Bit
d) Атрибут immutable
Правильный ответ: c

5. Какая команда изменит владельца файла 'document.txt' на пользователя 'admin'?
a) chmod admin document.txt
b) chown admin document.txt
c) chgrp admin document.txt
d) setuser admin document.txt
Правильный ответ: b

6. Что означает установленный бит SUID (Set User ID) на исполняемом файле?
a) Файл будет выполняться с правами владельца файла
b) Файл будет выполняться с правами группы файла
c) Файл может быть удален только суперпользователем
d) Файл будет зашифрован при выполнении
Правильный ответ: a

7. Какая команда установит права 'rw-r-----' на файл 'secret.txt' с использованием числовой нотации?
a) chmod 640 secret.txt
b) chmod 750 secret.txt
c) chmod u=rw,g=r,o= secret.txt
d) Варианты a и c верны
Правильный ответ: d

8. Какой командой можно просмотреть расширенные права доступа (ACL) файла 'shared.txt'?
a) getfacl shared.txt
b) ls -l shared.txt
c) acl show shared.txt
d) permissions shared.txt
Правильный ответ: a

9. Какой символ в выводе ls -l указывает, что на файле установлен бит SGID?
a) s в позиции прав группы
b) t в позиции прав других
c) s в позиции прав владельца
d) i в первой позиции
Правильный ответ: a

10. Что делает команда 'chmod g+s directory/'?
a) Устанавливает SUID бит на каталоге
b) Устанавливает Sticky Bit на каталоге
c) Устанавливает SGID бит на каталоге
d) Добавляет право записи для группы
Правильный ответ: c

11. Какой атрибут файла, управляемый командой 'chattr', делает файл неизменяемым даже для root?
a) +a (append only)
b) +i (immutable)
c) +d (no dump)
d) +s (secure deletion)
Правильный ответ: b

12. Как добавить пользователю 'ivanov' право на чтение файла 'report.txt' через ACL?
a) setfacl -m u:ivanov:r report.txt
b) setfacl -m user:ivanov:read report.txt
c) setfacl -a u:ivanov:r-- report.txt
d) acl add user ivanov r report.txt
Правильный ответ: a

13. Какая команда покажет атрибуты файла 'important.log'?
a) lsattr important.log
b) chattr -l important.log
c) getfattr important.log
d) attrib important.log
Правильный ответ: a

14. Что означает маска прав доступа (umask) 027?
a) Новые файлы создаются с правами 640, каталоги с 750
b) Новые файлы создаются с правами 750, каталоги с 640
c) Новые файлы создаются с правами 777, каталоги с 777
d) Запрещает создание новых файлов
Правильный ответ: a

15. Какой командой установить ACL-право, чтобы все новые файлы в каталоге 'uploads' наследовали права группы-владельца?
a) setfacl -d -m g::rwx uploads/
b) chmod g+s uploads/
c) setfacl -m d:g::rwx uploads/
d) Варианты a и c верны
Правильный ответ: d

16. Что означает символ 't' в выводе прав доступа каталога '/tmp' (drwxrwxrwt)?
a) Установлен бит SUID
b) Установлен бит SGID
c) Установлен Sticky Bit
d) Каталог временный
Правильный ответ: c

17. Как удалить все ACL записи для файла 'data.txt', оставив только базовые права?
a) setfacl -b data.txt
b) setfacl -x all data.txt
c) acl --clear data.txt
d) chmod 644 data.txt
Правильный ответ: a

18. Какое право доступа необходимо для переименования файла в каталоге?
a) Чтение (r) на каталоге
b) Запись (w) на каталоге
c) Выполнение (x) на каталоге
d) Запись (w) на самом файле
Правильный ответ: b

19. Какой командой рекурсивно изменить владельца всех файлов в каталоге '/data' на 'www-data'?
a) chown -R www-data /data
b) chown --recursive www-data /data
c) chmod -R www-data /data
d) Варианты a и b верны
Правильный ответ: d

20. Как установить атрибут 'append only' (только дополнение) на файл 'audit.log'?
a) chattr +a audit.log
b) chattr +i audit.log
c) setfacl -m a::append audit.log
d) chmod +a audit.log
Правильный ответ: a

21. Что делает команда 'chmod u=rwx,g=rx,o= script.sh'?
a) Устанавливает права 750 на файл
b) Устанавливает права 740 на файл
c) Удаляет все права у других пользователей
d) Варианты a и c верны
Правильный ответ: d

22. Как просмотреть ACL по умолчанию для каталога 'projects'?
a) getfacl -d projects
b) getfacl projects | grep default
c) ls -ld projects
d) Варианты a и b верны
Правильный ответ: d

23. Какая команда установит право выполнения файла 'program' только для владельца и группы?
a) chmod ug+x program
b) chmod u+x,g+x program
c) chmod 770 program
d) Варианты a и b верны
Правильный ответ: d

24. Какой числовой код соответствует установке SUID бита?
a) 1000
b) 2000
c) 4000
d) 8000
Правильный ответ: c

25. Как удалить ACL право для группы 'developers' из файла 'source.c'?
a) setfacl -x g:developers source.c
b) setfacl -m g:developers:- source.c
c) setfacl --remove-group developers source.c
d) acl del group developers source.c
Правильный ответ: a

26. Что означает право записи (w) для каталога?
a) Возможность редактировать файлы в каталоге
b) Возможность создавать и удалять файлы в каталоге
c) Возможность изменять права доступа каталога
d) Возможность читать содержимое каталога
Правильный ответ: b

27. Как установить права 2755 на каталог 'shared' (SGID + rwxr-xr-x)?
a) chmod 2755 shared
b) chmod g+s,755 shared
c) chmod u=rwx,g=rwxs,o=rx shared
d) Все перечисленные верны
Правильный ответ: d

28. Какой командой можно увидеть ACL в компактном формате?
a) getfacl -c file
b) getfacl -s file
c) getfacl --omit-header file
d) ls -l file
Правильный ответ: c

29. Что произойдет, если на каталоге установлен SGID бит?
a) Все новые файлы в каталоге унаследуют группу-владельца каталога
b) Все файлы в каталоге будут выполняться с правами группы
c) Только владелец сможет удалять файлы в каталоге
d) Каталог станет доступен только для членов группы
Правильный ответ: a

30. Какой атрибут позволяет только дополнять файл, но не изменять или удалять существующее содержимое?
a) immutable (+i)
b) append only (+a)
c) no dump (+d)
d) synchronous updates (+S)
Правильный ответ: b
