Работа с файлами в ОС Astra Linux
1. Согласно иерархии файловой системы Linux (FHS), в каком каталоге размещаются исполняемые файлы системных утилит, доступные всем пользователям?
a) /bin
b) /sbin
c) /usr/bin
d) Все перечисленные верны, но с разным назначением
Правильный ответ: d

2. Что такое индексный дескриптор (inode) файла?
a) Имя файла
b) Структура данных, хранящая метаданные файла (права, владелец, временные метки), но не имя
c) Содержимое файла
d) Путь к файлу в каталоге
Правильный ответ: b

3. Какой тип файла обозначается символом - в первой позиции вывода команды ls -l?
a) Обычный файл (regular file)
b) Каталог (directory)
c) Символическая ссылка (symbolic link)
d) Специальный файл устройства (device file)
Правильный ответ: a

4. Какой каталог согласно FHS является точкой монтирования для съемных носителей, таких как USB-флешки и DVD-диски?
a) /mnt
b) /media
c) /removable
d) /mount
Правильный ответ: b

5. Команда для перемещения на один каталог вверх (к родительскому каталогу) в командной строке:
a) cd up
b) cd /
c) cd ..
d) cd ~
Правильный ответ: c

6. Как создать несколько вложенных каталогов одной командой, например, dir1/dir2/dir3?
a) mkdir dir1 dir2 dir3
b) mkdir -p dir1/dir2/dir3
c) mkdir --recursive dir1/dir2/dir3
d) mkdir -v dir1/dir2/dir3
Правильный ответ: b

7. Какая команда используется для копирования файлов и каталогов с сохранением прав доступа и атрибутов?
a) cp -a
b) cp -r
c) cp -p
d) cp --preserve
Правильный ответ: a

8. Как найти все файлы с расширением .conf в каталоге /etc и его подкаталогах?
a) find /etc -type f -name "*.conf"
b) locate "*.conf" /etc
c) grep -r "\.conf$" /etc
d) ls -R /etc/*.conf
Правильный ответ: a

9. В Midnight Commander (mc) для пометки файла используется клавиша:
a) Пробел
b) Insert
c) Tab
d) F3
Правильный ответ: b

10. Какой каталог содержит конфигурационные файлы общесистемного назначения?
a) /home
b) /var
c) /etc
d) /usr
Правильный ответ: c

11. Как создать пустой файл с именем report.txt, если он не существует, или обновить его временную метку, если он существует?
a) nano report.txt
b) touch report.txt
c) echo "" > report.txt
d) cat > report.txt
Правильный ответ: b

12. Какой командой можно удалить каталог data вместе со всем его содержимым?
a) rm data
b) rm -r data
c) rmdir data
d) rm --recursive data
Правильный ответ: b

13. Как определить тип файла (например, текст, исполняемый, архив) с помощью команды file?
a) file -t filename
b) file filename
c) type filename
d) ls -l filename
Правильный ответ: b

14. В каком каталоге обычно размещаются домашние каталоги пользователей?
a) /home
b) /root
c) /usr/home
d) /var/home
Правильный ответ: a

15. Как создать жесткую ссылку (hard link) link.txt на файл original.txt?
a) ln -s original.txt link.txt
b) ln original.txt link.txt
c) cp original.txt link.txt
d) link original.txt link.txt
Правильный ответ: b

16. Какой командой можно подсчитать количество строк, слов и символов в файле document.txt?
a) count document.txt
b) wc document.txt
c) wc -l document.txt
d) stat document.txt
Правильный ответ: b

17. В Midnight Commander для просмотра файла без возможности редактирования используется клавиша:
a) F3
b) F4
c) F5
d) F6
Правильный ответ: a

18. Какой каталог предназначен для хранения изменяющихся файлов: логов, кэшей, очередей?
a) /tmp
b) /var
c) /run
d) /cache
Правильный ответ: b

19. Как переместить файл file.txt из текущего каталога в каталог /tmp?
a) cp file.txt /tmp
b) mv file.txt /tmp
c) move file.txt /tmp
d) rm file.txt /tmp
Правильный ответ: b

20. Как найти все файлы, размер которых превышает 100 МБ, в корневой файловой системе?
a) find / -size +100M
b) find / -size 100M
c) find / -size >100M
d) du -sh / --threshold=100M
Правильный ответ: a

21. Какой командой можно разбить вывод команды ls -l на страницы для постраничного просмотра?
a) ls -l | more
b) ls -l | less
c) ls -l | pg
d) Любой из перечисленных
Правильный ответ: d

22. Какой каталог содержит временные файлы, которые могут быть удалены без уведомления (например, при перезагрузке)?
a) /var/tmp
b) /tmp
c) /cache
d) /run
Правильный ответ: b

23. В Midnight Commander для копирования отмеченных файлов используется клавиша:
a) F5
b) F6
c) F7
d) F8
Правильный ответ: a

24. Как изменить владельца файла script.sh на пользователя admin?
a) chmod admin script.sh
b) chown admin script.sh
c) chgrp admin script.sh
d) setuser admin script.sh
Правильный ответ: b

25. Какой каталог содержит специфичные для системы исполняемые файлы, необходимые для загрузки и восстановления?
a) /bin
b) /sbin
c) /usr/sbin
d) /usr/local/sbin
Правильный ответ: b

26. Как создать символическую ссылку link на каталог /usr/share/doc?
a) ln -s /usr/share/doc link
b) ln /usr/share/doc link
c) link -s /usr/share/doc link
d) mklink /usr/share/doc link
Правильный ответ: a

27. Какой командой можно найти все файлы, измененные за последние 5 дней, в каталоге /var/log?
a) find /var/log -mtime -5
b) find /var/log -ctime 5
c) find /var/log -atime +5
d) ls -lt /var/log
Правильный ответ: a

28. Какой каталог содержит файлы устройств (device files), представляющие аппаратные устройства?
a) /sys
b) /proc
c) /dev
d) /run
Правильный ответ: c

29. В Midnight Commander для переименования или перемещения файла используется клавиша:
a) F5
b) F6
c) F7
d) F8
Правильный ответ: b

30. Какой командой можно сравнить два файла и вывести различающиеся строки?
a) diff file1 file2
b) cmp file1 file2
c) comm file1 file2
d) patch file1 file2
Правильный ответ: a
