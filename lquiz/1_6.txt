Работа с текстовой информацией в ОС Astra Linux
1. Какой оператор перенаправления добавит вывод команды (stdout) в конец файла log.txt, не удаляя существующее содержимое?
a) command > log.txt
b) command >> log.txt
c) command 2> log.txt
d) command &> log.txt
Правильный ответ: b

2. Что делает конструкция 2>&1 в команде command > file 2>&1?
a) Перенаправляет stderr (поток ошибок) в stdout, а затем stdout — в файл
b) Перенаправляет stdout в stderr, а затем stderr — в файл
c) Записывает оба потока в разные файлы
d) Закрывает поток ошибок
Правильный ответ: a

3. Какая команда позволяет просматривать длинный текстовый файл с возможностью прокрутки вперед/назад и поиска?
a) cat
b) less
c) head
d) more
Правильный ответ: b

4. Какой символ используется для конвейера (pipe), передающего стандартный вывод одной команды на стандартный ввод другой?
a) >
b) |
c) <
d) &
Правильный ответ: b

5. Какая команда выведет последние 10 строк файла /var/log/syslog?
a) head -10 /var/log/syslog
b) tail -10 /var/log/syslog
c) cat -n 10 /var/log/syslog
d) grep -A 10 /var/log/syslog
Правильный ответ: b

6. Какое регулярное выражение соответствует строке, содержащей только цифры?
a) ^[0-9]*$
b) ^[[:digit:]]+$
c) ^[0-9]+$
d) Все перечисленные верны
Правильный ответ: d

7. Как с помощью grep найти строки, содержащие слово "error", игнорируя регистр букв?
a) grep error file
b) grep -i error file
c) grep -v error file
d) grep --ignore error file
Правильный ответ: b

8. Какая команда sed заменит первое вхождение "foo" на "bar" в каждой строке файла input.txt?
a) sed 's/foo/bar/' input.txt
b) sed 's/foo/bar/g' input.txt
c) sed 's/bar/foo/' input.txt
d) sed 's/foo/bar/1' input.txt
Правильный ответ: a

9. В текстовом редакторе vim как перейти из командного режима в режим вставки?
a) Нажатие i
b) Нажатие Esc
c) Нажатие :
d) Нажатие a
Правильный ответ: a

10. Какой командой можно отсортировать строки файла data.txt по алфавиту и вывести результат на экран?
a) sort data.txt
b) uniq data.txt
c) order data.txt
d) cat data.txt | sort
Правильный ответ: a

11. Как перенаправить стандартный ввод (stdin) команды wc -l из файла list.txt?
a) wc -l < list.txt
b) wc -l > list.txt
c) wc -l | list.txt
d) list.txt | wc -l
Правильный ответ: a

12. Какая команда выведет уникальные строки из отсортированного файла (удалив дубликаты)?
a) sort -u file
b) uniq file (требует предварительной сортировки)
c) unique file
d) Варианты a и b верны
Правильный ответ: d

13. Какой оператор в регулярных выражениях означает "любой одиночный символ"?
a) *
b) ?
c) .
d) +
Правильный ответ: c

14. Как с помощью grep найти строки, НЕ содержащие шаблон "debug"?
a) grep -n debug file
b) grep -l debug file
c) grep -v debug file
d) grep --not debug file
Правильный ответ: c

15. Как в sed удалить все строки, содержащие "test", из файла?
a) sed '/test/d' file
b) sed 's/test//' file
c) sed '/test/p' file
d) sed 'd/test/' file
Правильный ответ: a

16. Как в редакторе vim сохранить изменения и выйти?
a) :q
b) :wq или :x или ZZ
c) :q!
d) :save
Правильный ответ: b

17. Какой простой текстовый редактор, часто устанавливаемый по умолчанию, может использоваться для базового редактирования в терминале?
a) vim
b) nano
c) emacs
d) gedit
Правильный ответ: b

18. Как вывести 5-ю строку файла config.conf?
a) head -5 config.conf | tail -1
b) sed -n '5p' config.conf
c) awk 'NR==5' config.conf
d) Все перечисленные способы работают
Правильный ответ: d

19. Что делает команда cut -d':' -f1 /etc/passwd?
a) Выводит все строки из файла
b) Выводит первое поле каждой строки, используя двоеточие как разделитель
c) Удаляет первое поле из каждой строки
d) Считает количество полей
Правильный ответ: b

20. Как в vim выполнить поиск слова "error" в прямом направлении?
a) /error
b) ?error
c) :find error
d) :grep error
Правильный ответ: a

21. Какая конструкция перенаправит stdout и stderr в разные файлы?
a) command > out.log 2> err.log
b) command &> out.log
c) command > out.log 2>&1
d) command 2> err.log 1> out.log
Правильный ответ: a

22. Как с помощью awk вывести второе поле каждой строки файла, разделенного запятыми?
a) awk '{print $2}' FS=',' file
b) awk -F',' '{print $2}' file
c) awk -F, '{print $2}' file
d) Все перечисленные верны
Правильный ответ: d

23. Какой командой можно посчитать количество строк, слов и символов одновременно?
a) wc
b) count
c) wc -lwc
d) lines
Правильный ответ: a

24. Как в nano сохранить файл с именем new.txt?

26. Как перенаправить stderr в "черную дыру" (полностью подавить ошибки)?

c) command &> /dev/null
d) command 2>&1 > /dev/null

29. Какой командой можно вывести строки файла, пронумеровав их?

30. Что делает команда tr 'a-z' 'A-Z' < file.txt?

c) Удаляет прописные буквы
