Процесс загрузки и выключения системы
1. Какая последовательность этапов загрузки ОС Linux является правильной для системы с UEFI?
a) POST → GRUB2 → Загрузка ядра → Initramfs → systemd
b) POST → Загрузка ядра → GRUB2 → Initramfs → systemd
c) POST → Загрузчик UEFI → GRUB2 → Ядро → Initramfs → systemd
d) POST → BIOS → MBR → GRUB → Ядро → Initrd → SysV init
Правильный ответ: c

2. Где хранится конфигурация GRUB2 в системах с UEFI?
a) В MBR первого диска
b) В файле /boot/grub/grub.cfg
c) В разделе EFI System Partition (ESP), обычно /boot/efi/EFI/astra/grub.cfg
d) В /etc/default/grub
Правильный ответ: c

3. Какой командой обновить конфигурацию GRUB2 после изменения файла /etc/default/grub?
a) update-grub
b) grub-mkconfig -o /boot/grub/grub.cfg
c) grub2-mkconfig -o /boot/efi/EFI/astra/grub.cfg
d) Любой из перечисленных, в зависимости от системы
Правильный ответ: d

4. Как добавить параметр quiet к загружаемому ядру в GRUB2?
a) В файле /etc/default/grub добавить в GRUB_CMDLINE_LINUX_DEFAULT
b) В файле /boot/grub/grub.cfg добавить в строку linux
c) В файле /etc/kernel/cmdline
d) Через grub-editenv
Правильный ответ: a

5. Какая команда загружает модуль ядра vfat в текущую сессию?
a) insmod vfat
b) modprobe vfat
c) loadmod vfat
d) module-load vfat
Правильный ответ: b

6. Какой командой сделать службу nginx автоматически запускаемой при загрузке системы?
a) systemctl enable nginx
b) systemctl start nginx
c) systemctl reload nginx
d) systemctl autostart nginx
Правильный ответ: a

7. Какой target в systemd соответствует уровню запуска с графическим интерфейсом?
a) multi-user.target
b) graphical.target
c) default.target
d) display-manager.target
Правильный ответ: b

8. Где размещаются пользовательские юниты systemd для создания собственной службы?
a) /etc/systemd/system/
b) /lib/systemd/system/
c) /usr/lib/systemd/system/
d) /run/systemd/system/
Правильный ответ: a

9. Как в Astra Linux SE запустить службу с определенным уровнем доверия (мандатным атрибутом)?
a) Использовать systemctl start service@trustlevel
b) В конфигурации юнита указать Parsectx= или использовать s утилиту в скрипте запуска
c) Изменить уровень доверия всей системы перед запуском
d) Запустить из сессии с нужным уровнем доверия
Правильный ответ: b

10. Какой файл содержит параметры, передаваемые ядру при загрузке в Astra Linux?
a) /etc/kernel/postinst.d/
b) /proc/cmdline
c) /etc/default/grub (после генерации grub.cfg)
d) /boot/grub/grub.cfg
Правильный ответ: b (актуальные параметры можно посмотреть в /proc/cmdline, настройка через /etc/default/grub)

11. Какой командой можно посмотреть журнал загрузки системы (boot log)?
a) journalctl -b
b) dmesg
c) cat /var/log/boot.log
d) Все перечисленные
Правильный ответ: d

12. Как запретить загрузку модуля ядра usb_storage?
a) Добавить blacklist usb_storage в файл /etc/modprobe.d/blacklist.conf
b) Выполнить rmmod usb_storage
c) В /etc/default/grub добавить modprobe.blacklist=usb_storage
d) Варианты a и c верны

13. Какой командой перейти в режим восстановления (single-user mode) без пароля?

c) В меню GRUB2 добавить systemd.unit=emergency.target

a) /lib/modules/$(uname -r)/
b) /usr/lib/modules/
c) /boot/modules/
d) /etc/modules-load.d/

a) systemctl reboot
b) systemctl poweroff

Правильный ответ: a

16. Какой раздел должен быть смонтирован как /boot/efi для правильной работы UEFI?
