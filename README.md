# Quiz Application

Веб-приложение для создания и прохождения интерактивных тестов (квизов) для сотрудников.

## Технологический стек

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python Flask
- **База данных**: SQLite

## Функциональность

### 1. Создание тестов
- Форма для создания новых тестов с указанием названия
- Управление тестами (создание, редактирование, удаление)

### 2. Управление вопросами и ответами (CRUD)
- Динамическое добавление любого количества вариантов ответов к каждому вопросу
- Редактирование вопросов на отдельных страницах
- Автоматическое сохранение данных при переходе между вопросами

### 3. Навигация по вопросам
- Пошаговое редактирование вопросов
- Переход к следующему/предыдущему вопросу с сохранением данных
- Интуитивный интерфейс управления контентом теста

## Структура базы данных

### Основные сущности:
- **Тесты** - информация о тестах
- **Вопросы** - вопросы в рамках тестов
- **Варианты ответов** - варианты ответов для каждого вопроса

## Установка и запуск

1. Установите зависимости:
```bash
pip install flask psycopg2
```

2. Настройте базу данных PostgreSQL

3. Запустите приложение:
```bash
python app.py
```

4. Откройте в браузере: `http://localhost:5000`

## Использование

1. Создайте новый тест через соответствующую форму
2. Добавляйте вопросы к тесту
3. Для каждого вопроса создавайте варианты ответов
4. Используйте навигацию для перемещения между вопросами
5. Данные автоматически сохраняются при переходе

## Особенности

- Динамический интерфейс без перезагрузки страниц
- Автосохранение прогрередактирования
- Гибкая система управления вариантами ответов
- Простое добавление новых вопросов и ответов
